# Second Brain — an Obsidian vault

A complete personal knowledge system: capture everything, organise it with PARA, think in linked atomic notes (Zettelkasten), and review on a daily/weekly/monthly rhythm. Built to pair with ScaleMyLife — the app is your **engine of action**, this vault is your **memory and thinking space**.

## Quick start (5 minutes)

1. Install [Obsidian](https://obsidian.md) (free, all platforms).
2. **Open folder as vault** → select this `second-brain` folder.
3. Open **[[Home]]** — it's the dashboard and explains everything.
4. Press `Ctrl/Cmd+P` → "Daily note" to create today's note and start.

Optional but recommended: install the community plugins listed in [[90 - Meta/Plugins|Plugins]] (Dataview makes the Home dashboard live; Templater and Tasks supercharge the templates). Everything works without them — the plugins just automate what you'd do by hand.

## The folder map

| Folder | What lives here |
|---|---|
| `00 - Inbox` | Everything you capture, before it's processed. Aim for empty. |
| `01 - Daily` | One note per day: intentions, log, journal. |
| `02 - Projects` | Short-term efforts with a goal and a deadline. |
| `03 - Areas` | Long-term responsibilities with a standard to maintain (Health, Work…). |
| `04 - Resources` | Topics of interest, source notes on books/articles/videos. |
| `05 - Zettelkasten` | Permanent notes — one idea each, in your own words, densely linked. |
| `06 - Archive` | Anything inactive. Never delete, archive. |
| `07 - Templates` | Templates for every note type. |
| `90 - Meta` | How the system works, workflows, plugin guide. |
| `99 - Attachments` | Images and files (set as the default attachment folder). |

## The method in one paragraph

**Capture** anything that resonates into the Inbox with zero friction. **Organise** it during your daily/weekly review by asking "where will I *use* this?" (a project → Projects, a responsibility → Areas, future reference → Resources, nowhere → Archive or delete). **Distil** what matters into atomic permanent notes in the Zettelkasten, linked to what you already know. **Express** — use those notes to write, build, decide and act. Full details in [[90 - Meta/How this vault works|How this vault works]] and [[90 - Meta/Workflows|Workflows]].
