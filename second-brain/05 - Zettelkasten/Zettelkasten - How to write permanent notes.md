# 🕸️ Zettelkasten — how to write permanent notes

This folder is the actual *brain* part of the second brain: **permanent notes** — ideas that are yours, written to be understood in ten years, connected into a web. Everything else in the vault feeds this.

## The rules (there are only four)

1. **Atomic** — one idea per note. If you write "and another thing", that's note #2.
2. **In your own words** — no copy-paste. If you can't rephrase it, you don't understand it yet.
3. **Claim as title** — the title is a full statement, not a topic. "Habits" is a bad title; [[Atomic notes make ideas reusable]] is a good one. You can then *read* your links as sentences.
4. **Always linked** — before saving, ask "what does this connect to, support, or contradict?" and link at least one existing note. An unlinked note is a leaflet in a drawer.
