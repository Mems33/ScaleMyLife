# Atomic notes make ideas reusable

An idea captured as one self-contained note can be linked from any future context — a project, an essay, a decision. The same idea buried in a 2,000-word "Book notes: XYZ" file can only ever be found by rereading that file.

Atomicity is what turns notes from *storage* into *building blocks*: big pieces of writing and thinking get assembled from small notes, the way functions compose into programs.

The test: could I link to this note from a completely different topic and have it still make sense? If not, split it.
